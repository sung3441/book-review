# 클래스 추출

너무 많은 일을 하고 있는 클래스에서 책임을 분리할 수 있는 필드와, 메서드를 추출해서 새로운 클래스를 만든다.

### 클래스 추출 방법

1. 새로운 클래스 작성 
   1. 필드와 메서드(책임)를 어떻게 추출할지 결정
   2. 추출한 필드와 메서드를 담당할 새로운 클래스 작성
   3. 기존 클래스에서 새로운 클래스로 링크 작성
      - 링크는 가능한 한 단방향
2. 필드 이동
3. 메서드 이동
4. 추출한 클래스 검토
   1. 클래스 인터페이스(API)를 줄일 수 있는가
   2. 새로운 클래스를 외부에 공개해야 되는지
   3. 공개한다면 외부에서 수정 가능하게 할 것인가

### 클래스 추출의 장점

- 기능 추가의 번잡함이 줄어듦
- 기능 추가가 더욱 적절한 클래스 안에서 이루어짐
